HTML Laboratory

HTML (HyperText Markup Language) is the standard markup language used to create web pages and web applications. It is used to structure content on the internet such as text, images, tables, and links.

Topics Covered in HTML Lab

  1. Basic HTML structure
  2. Text formatting tags
  3. Lists (ordered, unordered, definition)
  4. Images and hyperlinks
  5. Tables and table attributes
